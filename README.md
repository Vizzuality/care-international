# CARE International

## Installation

Requirements:

* [NodeJs](https://nodejs.org/es/download/)

To install run command:

```bash
npm install
```

## Usage

In development mode, we can start a local server running:

```bash
npm start
```

To build in production mode we can use:

```bash
npm run build
```

To publish in gh-pages (staging mode):

```bash
npm run publish
```
